Method

The page keeps the story compact: low precision for the initializer, high precision for the final numerical contract.

\[ A \xrightarrow{\mathrm{scale+quantize}} A_8,\qquad G_8 = A_8^{T}A_8,\qquad R_0 = \operatorname{chol}(G_8 + \varepsilon I),\qquad Q_0 = A R_0^{-1}. \]
01 / initializer

INT8 Gram start

Use INT8/INT32 throughput to build a cheap approximation of the Gram matrix before refinement.

02 / correction

FP64 polishing

Correct the basis with high-precision residual information instead of pretending INT8 is enough.

03 / safeguard

Shifted variants

When the pivot path becomes fragile, shifted Cholesky variants keep the factorization alive.

\[ \norm{Q^{T}Q-I}_F \quad\text{is the visible contract;}\quad \cond{A}^{2} \quad\text{is the hidden tax.} \]
